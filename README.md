# AdventOfCyber

A structured repository documenting my yearly participation in the TryHackMe Advent of Cyber event.

This repository serves as my long-term cybersecurity learning archive, containing:

* notes
* methodologies
* writeups
* scripts
* lessons learned
* defensive and offensive security concepts

The goal is not just to complete challenges, but to deeply understand the technologies, attack paths, and security principles behind them.

* Malware Analysis
* SIEM & Log Analysis

* Incident Response
